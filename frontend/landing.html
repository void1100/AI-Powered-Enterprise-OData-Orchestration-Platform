Features

Everything you need to turn raw OData services into actionable insights.

Handle missing values, remove outliers, normalize, encode — all before ML analysis.

Beautiful dark and light themes. Works on desktop, tablet, and mobile.

MCP Protocol

Industry-standard Model Context Protocol. Integrate with any MCP-compatible client.

How It Works

Three steps from question to insight.

1

Ask in Plain English

Type a natural language question about your data.

"Show top 5 customers from Germany"
2

LLM Generates the Query

The reasoning engine creates an optimized OData plan.

{"entity":"Customers","filter":"Country eq 'Germany'","top":5}
3

Get Results + Insights

View data as table, charts, or run ML analysis for deeper insights.
